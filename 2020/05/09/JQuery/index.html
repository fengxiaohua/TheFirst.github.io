5、DOM对象

DOM对象是由浏览器创建的

<div>1</div>    domObj_1
<div>2</div>    domObj_2

DOM对象在浏览器管理内存中以 二叉树存在(document树)

jQuery对象与DOM对象

var $obj = $("#one");
var value = $obj.val();
$obj.val("Hello" + value);

DOM对象：由浏览器创建的对象，一个HTML对应一个DOM对象

jQuery对象：由jQuery函数$()创建的对象，本质是一个数组，保存了当前被定为的所有DOM对象

互相转换：

  1. 如何将DOM转换为jQuery（放到数组里去）

  2. jQuery转DOM（从数组里面拿出来）

    var $obj = $(":text");
for(var i = 0; i < $obj.length; i++){
    var domObj = $obj.get(i);

jQuery如何定为DOM对象

使用 选择器（3） 和 过滤器（6） 进行定为

选择器

1、基本选择器

根据标签ID，标签类型，标签样式选择器进行定位

$(“#xxx”)

$(“xxx”)

$(“.xxx”)

扩展：

全部定位：$(“*”)

组合条件选择：$(“条件1,条件2”) 这里,相当于and

2、 层级选择器

根据HTML标签之间的 兄弟关系（同级） 和 父子关系（包含） 进行定位

定位满足条件的子标签

一、定位满足条件的子标签

$(“定位父标签条件>子标签定位条件”)：定位当前父标签下所有满足条件的 直接子标签

$(“定位父标签条件 子标签定位条件”)：定位当前父标签下所有满足条件的直接子标签 和 间接子标签

二、定位满足条件的兄弟标签

$(“定位当前标签条件+条件”)：定位当前标签后面与之紧邻并且满足定位条件的兄弟标签

$(“定位当前标签条件~条件”)：定位当前标签后面，所有满足条件的兄弟标签

$(“定位当前标签条件”).siblings(“条件”)：定位当前标签的前面与后面所有满足条件的兄弟标签

1、 val函数：

1）$obj.val(值)：对当前jQuery对象中所有的DOM对象的value同一赋值

2）$obj.val()：读取当前jQuery对象中第一个DOM对象的value属性值

如果要读第二个呢？（循环）

function fun1(){
    var $obj = $(":radio");
    for(var i = 0; i < $obj.length; i++){
        var value = $obj.get(i).value;
        alert("第"+ i + "个radio的value=" + value);
    }
}

以上写法很low，都成DOM对象了

—————->each函数，用于遍历jQuery对象中保存的所有DOM对象（底层还不是转成DOM对象做的。。。）

2、each函数

$obj.each( function (下标值,DOM对象){ 操作 } )

function fun3(){
    var $obj = $(":radio");
    $obj.each(

            alert("第" + i + "个value=" +domObj.value);

3、text函数：

1）$obj.text(“文字显示内容”)：对当前jQuery对象中所有的DOM对象的【文字显示内容】进行同一赋值

2）$obj.text()：读取当前jQuery对象中第一个DOM对象的【文字显示内容】

————->代替innerText

4、attr函数

1）$obj.attr(“基本属性名”,”值”)：对当前jQuery对象中所有的DOM对象的【指定基本属性】进行同一赋值

2）$obj.attr(“基本属性名”)：读取当前jQuery对象中第一个DOM对象的【指定基本属性】

5、prop函数

1）$obj.prop(“工作状态属性”,boolean值)：对当前jQuery对象中所有的DOM对象的【指定工作状态】进行同一赋值

2）$obj.prop(“工作状态属性”)：读取当前jQuery对象中第一个DOM对象的【指定工作状态】值
